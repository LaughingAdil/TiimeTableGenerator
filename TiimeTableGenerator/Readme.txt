
**** SUMMARY ****
The Class Schedule Management System is a web application designed to simplify the organization and management of class schedules within educational institutions. 
The logic used in this application is performed on the client side using raw JavaScript and C#. 
It operates with an in-memory database, meaning it does not remember user preferences or generated data between sessions.

**** USAGE ****
Clone or download the repository to your local machine.
Open the application in your preferred web browser using the link in the Technical Document PDF.
